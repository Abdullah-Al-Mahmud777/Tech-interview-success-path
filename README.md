**Tech Interview Success Path** is a structured learning repository designed for **Computer Science students and Full-Stack developers** preparing for software engineering interviews.

This repository covers:

- Computer Science fundamentals
- Full-Stack development
- Data Structures & Algorithms
- System Design
- Real-world project building
